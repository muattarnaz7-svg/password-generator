
# Password Generator 🔐

A simple Python script that generates a secure random password using letters and numbers.

## Features
- Asks the user for desired password length
- Generates a random password using uppercase, lowercase letters, and numbers
- Uses Python's `secrets` module for cryptographically secure randomness

## Technologies Used
- Python 3
- `secrets` module
- `string` module

## How to Run

```bash
python password.py
```
